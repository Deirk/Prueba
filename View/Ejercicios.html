Descomposición

¡Vamos a descomponer!

La descomposición es una habilidad importante para crear algoritmos y procesos a la hora de programar. Consiste en hacer un problema muy complejo en partes más simples y manejables.
Tomate 5 minutos para realizar este ejercicio y ser un experto descomponiendo.

Abstracción

¡Vamos a razonar!

El razonamiento abstracto nos ayuda a resolver problemas lógicos, deduciendo que va a suceder a partir de una situación planteada.
En este caso, debemos descubrir y analizar cada uno de los siguientes ejercicios y resolver los interrogantes con las opciones dadas (a,b,c) a cada problema.

Generalización

¡Diseñemos algoritmos!

Todos sabemos resolver problemas y lo hacemos constantemente, pero cuando programamos debemos dividir el problema en pequeñas partes o instrucciones para que el ordenador lo entienda.
Debemos se muy especificos para dar estas ordenes, pero antes, debemos ser unos expertos en ellas.

Depuración

¡Vamos a depumponer!

Si hay errores en el código, debemos usar nuestra habilidad deductiva para encontrar la causa. Para ello, necesitamos tener una visión general del problema y analizar cada paso o cada acción que realicemos. Aquí deberás jugar con números para formar una imagen de forma lógica.
